# BlogByBro

A personal blog built with Next.js and Supabase. Write and publish posts from a private admin panel; readers visit the public site.

---

## Tech Stack

| Layer              | Technology                       |
| ------------------ | -------------------------------- |
| Frontend + Backend | Next.js 16 (App Router)          |
| Database + Auth    | Supabase (PostgreSQL)            |
| Image Storage      | Supabase Storage                 |
| Styling            | Tailwind CSS + Typography plugin |
| Rich Text Editor   | TipTap                           |
| Hosting            | Vercel                           |
